# approvalflow 表单流程设计引擎
```
This is a form process design engine
```

### What can he do
> 1.该引擎能够帮助你低成本、快速的实现表单设计、流程设计的前端功能
> 2.你可以通过该引擎扩展一个页面设计器，来节约搭建前端的成本
